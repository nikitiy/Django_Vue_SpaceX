# Django_Vue_SpaceX

### Установка проекта
```
python3 -m venv venv
pip install -r backend/requirements.txt
python3 backend/manage.py migrate
python3 backend/manage.py createsuperuser
python3 backend/manage.py runserver

```

### Переходим в браузер
http://127.0.0.1:8000/

http://127.0.0.1:8000/admin - админ панель
